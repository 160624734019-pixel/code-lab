# PulseGuard AI - Blood Pressure Prediction System

An AI-powered web application that predicts hypertension risk and provides personalized health recommendations.

## Features

- **ML Classification**: Hypertension risk prediction (normal / elevated / hypertension stage 1-2)
- **PulseGuard Risk Score**: 0-100 scale personal risk meter with explainable factors
- **BP Trend Forecasting**: LSTM-based forecast of systolic/diastolic 6 months ahead
- **Explainable AI (SHAP)**: Shows which factors drive your risk
- **Personalized Recommendations**: Tailored diet, exercise, and monitoring advice
- **Rural-friendly & Scalable**: Low-bandwidth, works with basic smartphones

## How to Use

1. Open `index.html` in any modern web browser
2. Navigate to the demo section
3. Enter health data (age, BMI, blood pressure, etc.)
4. Click "Predict Risk" to see your assessment
5. Review risk factors and personalized recommendations

## Technology Stack

- Python
- Scikit-learn
- TensorFlow
- Flask / FastAPI
- Streamlit
- SHAP
- Pandas
- XGBoost
